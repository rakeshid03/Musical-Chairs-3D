# Musical Chairs 3D

> **Run. Listen. Rush. Sit. Become the Champion! 🪑🏆**

Musical Chairs 3D is a fast-paced casual browser game. Players run around a circle while music plays, then race to grab a chair the moment the music stops. Outsmart AI opponents, unlock skins, earn coins, and become the ultimate champion.

# 🎯 Gameplay

1. Walk around the chairs while music plays.
2. Listen carefully.
3. When the music stops...
4. Run to the nearest chair.
5. Players without chairs are eliminated.
6. One chair disappears every round.
7. Last player sitting wins.

---

# 🕹 Controls

### Mobile

- Left Joystick → Move
- SIT Button → Sit on nearby chair

### Desktop

- WASD / Arrow Keys → Move
- Space → Sit
